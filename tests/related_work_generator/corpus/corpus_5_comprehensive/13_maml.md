# Model-Agnostic Meta-Learning for Fast Adaptation of Deep Networks

**Authors:** Chelsea Finn, Pieter Abbeel, Sergey Levine
**Year:** 2017
**Venue:** ICML
**DOI:** 10.48550/arxiv.1703.03400

## Abstract

We propose an algorithm for meta-learning that is compatible with any model trained with gradient descent. MAML trains the model to be easy to fine-tune, requiring only a small number of gradient steps to achieve good performance on a new task, including classification, regression, and reinforcement learning.
