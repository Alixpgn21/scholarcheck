# You Only Look Once: Unified, Real-Time Object Detection

**Authors:** Joseph Redmon, Santosh Divvala, Ross Girshick, Ali Farhadi
**Year:** 2016
**Venue:** CVPR
**DOI:** 10.1109/cvpr.2016.91

## Abstract

We present YOLO, a new approach to object detection. We reframe object detection as a single regression problem from image pixels to bounding box coordinates and class probabilities. A single neural network predicts bounding boxes and class probabilities directly from full images in one evaluation.
