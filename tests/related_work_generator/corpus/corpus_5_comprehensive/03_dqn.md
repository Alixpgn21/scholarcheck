# Human-level control through deep reinforcement learning

**Authors:** Volodymyr Mnih, Koray Kavukcuoglu, David Silver
**Year:** 2015
**Venue:** Nature
**DOI:** 10.1038/nature14236

## Abstract

We present a deep reinforcement learning method capable of human-level performance in Atari games. The algorithm uses a deep neural network to approximate the Q-function, trained with experience replay and a target network to stabilize learning.
